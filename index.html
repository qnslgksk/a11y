ARIA는 역할(role), 상태(state), 속성(property)으로 보조기술로 스크린리더 사용자들의 비시각적 경험을 제공한다.

role을 주었다고 키보드동작이나 스타일을 제공하지 않으므로 스크립트 추가 작업 필요 잘못된 태그를 사용하여 aria-*를 남발하면 안하느니만 못함

3. 알게된 점

aria-label, aria-labelledby, aria-description, aria-describedby속성을 주는 요소에 설명을 제공하는 속성 aria-label은 영역에 heading태그가 없을 때 사용
aria-labelledby는 heading태그,a, button요소를 id값으로 참조할 때 사용
aria-label과 aria-labelledby를 함께 사용하면 aria-label보다 aria-labelledby속성이 우선순위가 높음
그리고 숨김 처리한 설명을 참조하면 안되고 참조가 불가능한 경우 aria-label속성 사용 aria-description은 자세한 설명을 할 때 사용 aria-describedby는 설명글을 id값으로 참조 할 때 사용
